# phoneBook_flutter

# Техническое задание: Приложение "Телефонная книга"

## 1. Общее описание проекта

Разработать простое мобильное приложение на Flutter для управления контактами. Приложение предназначено для студентов начального уровня и должно демонстрировать базовые навыки работы с Flutter.

## 2. Функциональные требования

### 2.1. Главный экран (Список контактов)

**Обязательные элементы:**
- Заголовок "Телефонная книга" или "Мои контакты"
- Поле поиска контактов
- Список контактов
- Кнопка "Добавить контакт" (FloatingActionButton)
- Счетчик общего количества контактов

**Отображение контакта в списке:**
Каждый элемент списка должен содержать:
- Фото контакта (или инициалы в цветном круге, если фото нет)
- Имя и фамилия
- Номер телефона
- (Опционально) Иконка для быстрого звонка
- Свайп для удаления

**Логика работы:**
1. Отображение всех контактов в алфавитном порядке
2. Поиск по имени и номеру телефона в реальном времени
3. Нажатие на контакт открывает детальную информацию
4. Пустой список показывает сообщение "Нет контактов"

### 2.2. Экран добавления/редактирования контакта

**Обязательные элементы:**
- `AppBar` с заголовком "Новый контакт" или "Редактировать контакт"
- Поле для выбора/добавления фото (с кнопками "Камера" и "Галерея")
- Текстовое поле "Имя" (обязательное)
- Текстовое поле "Фамилия" (обязательное)
- Текстовое поле "Номер телефона" (обязательное)
- Текстовое поле "Email" (опционально)
- Текстовое поле "Заметки" (опционально)
- Кнопка "Сохранить"
- Кнопка "Отмена"

**Логика работы:**
1. Валидация полей (имя, фамилия и телефон обязательны)
2. Проверка формата номера телефона
3. Возможность добавить фото через камеру или галерею
4. Автоматическое форматирование номера телефона
5. Сохранение контакта в список

### 2.3. Экран детальной информации о контакте

**Обязательные элементы:**
- Фото контакта (большого размера)
- Полное имя
- Номер телефона
- Email (если указан)
- Заметки (если указаны)
- Кнопка "Позвонить" (имитация)
- Кнопка "Написать SMS" (имитация)
- Кнопка "Редактировать"
- Кнопка "Удалить"

**Логика работы:**
1. Отображение всей информации о контакте
2. Кнопка "Позвонить" имитирует звонок (показывает SnackBar)
3. Кнопка "Написать SMS" имитирует отправку сообщения (показывает SnackBar)
4. Редактирование открывает экран редактирования с заполненными полями
5. Удаление с подтверждением
